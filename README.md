Bank Loan Analysis Dashboard

This project analyzes bank loan data to evaluate lending performance and understand the overall health of a loan portfolio using Tableau.

Objective

The goal of this project is to:

Track key loan metrics over time
Analyze loan performance
Identify good and bad loans
Support data-driven decisions
Dashboards
Summary Dashboard
Total Loan Applications (MTD and MoM)
Total Funded Amount
Total Amount Received
Average Interest Rate
Average Debt-to-Income Ratio (DTI)
Good vs Bad Loan Analysis
Overview Dashboard
Monthly trends
Regional analysis
Loan term distribution
Employment length analysis
Loan purpose breakdown
Home ownership analysis
Details Dashboard
Detailed view of all loan data
Tools Used
Tableau
Excel / CSV
